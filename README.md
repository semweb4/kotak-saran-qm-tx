# Kotak Saran Digital QM-TX

Project Kotak Saran Digital untuk QM-TX.

## Fitur

- Formulir digital untuk提交 saran
- Integrasi dengan Google Sheets via Google Apps Script
- Responsive design dengan Bootstrap 5

## Cara Penggunaan

1. Buka `index.html` di browser
2. Isi nama dan saran Anda
3. Klik tombol Submit

## Teknologi

- HTML5
- Bootstrap 5
- Google Apps Script
